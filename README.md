# CoreXY Set v2

A compact, precision **CoreXY XY-stage** designed for **education, research, and digital fabrication**.  
This project emphasizes **mechanical correctness, alignment-aware assembly, and reproducibility**, making it suitable for student labs, prototyping platforms, and open-hardware distribution.
